# 🎬 TidianeFlix - Système de Billetterie Cinéma Intelligent

Application full-stack de gestion de billetterie pour cinéma avec **analytics avancés** et **machine learning** pour optimiser les opérations et maximiser les revenus.

## ✨ Fonctionnalités

### 📊 Analytics Avancés
- **KPIs financiers** en temps réel (revenus, billets vendus, prix moyen)
- **Segmentation clients** automatique (VIP ≥10K, Fidèle ≥5K, Occasionnel)
- **Top performers** (clients & films)
- **Tendances temporelles** interactives

### 🤖 Machine Learning
- 🔮 **Prédiction ventes** (7 jours futurs) par régression linéaire
- 🎬 **Recommandation films** par genre (content-based filtering)
- 💯 **Scoring clients** (probabilité de réachat)
- 📈 **Optimisation occupation** des salles

### 🎥 Gestion Opérationnelle
- Catalogue de 15 films
- 3 salles (350 places total)
- Base clients avec historique
- Vente de billets rapide

## 🎨 Interface
- Landing page style Netflix
- Graphiques interactifs Plotly
- Dark theme élégant
- Responsive design
